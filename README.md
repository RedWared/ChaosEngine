# ChaosEngine
ChaosEngine source code

## Build
```
mkdir build
cd build
cmake ..
make
```

## Example of use.
```C++
#include <ChaosEngine/engine.hpp>

int main(){
	Engine wah;
	wah.log("ChaosEngine basic example");
	wah.CreateWin();
	return 0;
}
```
